Non obtrusive plugin for adding input box controls for integer values

Feel free to use it and abuse it!

Example

Below code adds two buttons next to three regular <input> elements: one for increasing, and one for decreasing the value in each box 
  $(document).ready(function() {
    $('.demo_input').inputCtl({minVal: 0, maxVal: 10, step: 2});
  });


Usage

.inputCtl( options )

options Set of key/value pairs to configure the controls. All options are optional.

step Integer

Step to increase/decrease the input box value with. Default is 1

minVal Integer

Lower limit of input box value. Default is false, ie. no minimum value

maxVal Integer

Upper limit of input box value. Default is false, ie. no maximum value

incText String

Text to display in the increase button. Default is +

decText String

Text to display in the decrease button. Default is -
